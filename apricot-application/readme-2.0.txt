------------------------------------------
Apricot DB, Release 2.0, date: 27/11/2019
------------------------------------------
This is a feature release.
* The access to all major functions of the apllication has been provided via the toolbar;
* The function of comparison of the Snapshots was implemented;
* All found defects have been fixed.

This version of "Apricot DB" provides the Reverse Engineering of the following databases:
Oracle;
SQL Server;
MySQL;
PostgreSQL;
DB2;
H2

----------------------------
Apricot DB Installation
----------------------------
1. Unzip apricotdb-2.0-windows-x64.zip;
2. Run the application using the script: apricotdb\apricotdb.bat
