------------------------------------------
Apricot DB, Release 2.2 14/03/2020
------------------------------------------

* features...
- improved the error diagnostics;
- the "select related entities" functionality was extended with additional services;
- the "Quick View" was implemented;
- the native authentication for SQL Server was implemented;
- all the bugs found by now have been fixed.

This version of "Apricot DB" provides the Reverse Engineering of the following databases:
Oracle;
SQL Server;
MySQL;
PostgreSQL;
DB2;
DB2 LUW;
H2

----------------------------
Apricot DB Installation
----------------------------
1. Unzip apricotdb-2.2-windows-x64.zip;
2. Run the application using the script: apricotdb\apricotdb.bat
