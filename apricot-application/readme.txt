----------------------------
Apricot DB, Release 0.6a, 14/08/2019
----------------------------
The bugfix Release:
* the apricotdb.sh script for running Linux version of Apricot DB was fixed (thanks Gary Harris);
* the Reverse Engineering scanner of the Oracle Database was fixed (thanks Gary Harris and Simon Abi Saad)

For more details please refer to the user documentation: "Apricot DB User Guide v0.6.pdf".
This version of "Apricot DB" provides the Reverse Engineering of the following databases:
Oracle;
SQL Server;
MySQL;
PostgreSQL;
DB2;
H2

----------------------------
Apricot DB Installation
----------------------------
On Windows:
1. Unzip apricotdb-0.6a-windows-x64.zip;
2. Run the application using the script: apricotdb\apricotdb.bat

On Linux:
1. Unzip apricotdb-0.6a-linux-x64.tar.gz;
2. Run the application using the script: apricotdb/apricotdb.sh
