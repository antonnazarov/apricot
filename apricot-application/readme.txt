----------------------------
Apricot DB, Release 0.6, 06/08/2019
----------------------------
This release includes a new implementation of some important functions as well as the bugfixes.

For more details please refer to the user documentation: "Apricot DB User Guide v0.6.pdf".
This version of "Apricot DB" provides the Reverse Engineering of the following databases:
Oracle;
SQL Server;
MySQL;
PostgreSQL;
DB2;
H2

----------------------------
Apricot DB Installation
----------------------------
On Windows:
1. Unzip apricotdb-0.6-windows-x64.zip;
2. Run the application using the script: apricotdb\apricotdb.bat

On Linux:
1. Unzip apricotdb-0.6-linux-x64.tar.gz;
2. Run the application using the script: apricotdb/apricotdb.sh
